Process
How it works

Three simple steps to detect whether your media is real or AI-generated.

1

Upload your file

Drag and drop or click to select an image (JPG, PNG, WEBP) or a video (MP4, AVI, MOV). Maximum 100 MB for video.

2

Ensemble analysis

Three models independently analyze each frame: ResNet50 and two HuggingFace detectors. Their predictions are equally weighted (33% each) into a final verdict.

3

View detailed results

Get a clear verdict with per-model breakdowns, confidence scores, and a probability chart to understand how the decision was made.

Features
What this system can do

Built as a final year project with a focus on accuracy, explainability, and ease of use.

Image Detection

Analyzes images for AI-generation artifacts. Supports JPG, PNG, WEBP, and BMP up to 10 MB.

Video Detection

Samples frames across the full video length and plots AI probability per frame with a threshold line.

Per-model Breakdown

Each model in the ensemble votes independently. Results are shown per model so you can see where disagreements occur.

Analytics Dashboard

Tracks all past detections locally. View breakdowns by verdict type, file type, and scan volume over time.

Ensemble Voting

Three specialized models vote equally (33% each): ResNet50, AI-image-detector, and SDXL-detector. Their predictions are averaged for robust, balanced detection results.

Colab + Render

Flask backend runs on Google Colab with GPU support, exposed via ngrok. Frontend deploys on Render. Full setup with no build steps required.

About this project

AI-Driven Deepfake Detection is a deepfake and AI-generated image/video detector built as a final year B.Tech Computer Science & Engineering project (2025–26). It uses a 3-model ensemble: a custom-trained ResNet50 (97.61% accuracy) combined with two HuggingFace detectors (AI-image-detector & SDXL-detector). Each model contributes equally (33%) to the final verdict. The backend runs on Google Colab via Flask, exposed through ngrok, and the frontend deploys on Render for easy access without build steps.
